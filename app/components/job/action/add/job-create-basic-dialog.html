Create job

Adds a job to the selected account

Id is a required field
Id has a maximum length of 64 characters
Id can contain any combination of alphanumeric characters including hyphens and underscores.
Display name has a maximum length of 1024 characters
Priority values can range from -1000 to 1000
Retry count values can range from -1 to 100

Select a pool to run the job on

You must select a pool for this job to run on
