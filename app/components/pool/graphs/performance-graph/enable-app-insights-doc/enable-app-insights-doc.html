To view stats like cpu and memory usage you must enable application insights

You can check the documentation here https://github.com/Azure/batch-insights

Quick Steps

  1. 1. Add `APP_INSIGHTS_KEY` environment variable in the start task(App insights instrumentation key)
  2. 2. Add `APP_INSIGHTS_APP_ID` environment variable in the start task(App insights application ID)
  3. 3. Add os specific one liner in your start task see doc

Please note this feature is not yet built into the Azure Batch service. This feature is in beta in BatchLabs
